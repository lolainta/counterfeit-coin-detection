# Counterfeit Coin Detection

## Features
- Multithread
- Progress bar
- Solve by lasso
- Visualized result

## Usage

Install dependencies via pip3.
```bash
pip3 install -r requirements.txt
```

Run and wait for a while, the result will be generate in `result.png`.
```bash
python3 main.py
```
